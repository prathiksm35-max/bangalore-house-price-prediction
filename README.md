# 🏠 Bangalore House Price Prediction

## Overview
Predicts house prices in Bangalore using Machine Learning 
on a dataset of 13,000+ real properties.

## Results
| Model | R² Score |
|-------|----------|
| Linear Regression | 0.783 |
| Decision Tree | 0.973 |
| **Random Forest** | **0.987** |

✅ Best Model: Random Forest  
✅ MAE: ₹2.16 Lakhs  

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

## Project Steps
1. Data Loading & Exploration
2. Data Cleaning & Missing Value Treatment
3. Feature Engineering (price_per_sqft)
4. Outlier Removal
5. Model Training & Comparison
6. Model saved using Pickle

## Dataset
Bangalore House Price Data — 13,000+ properties  
Source: Kaggle
